## 🛠️ Tech Stack
- **Frontend:** Next.js 15, React, Tailwind CSS
- **Backend (for contact form & API):** Flask
- **Database:** PostgreSQL (for backend projects)

- **Tools & Libraries:** React Icons, Formspree
